# Private Board AI

> 難しい意思決定のための、あなた専用の役員会。

Private Board AI は、個人的あるいは戦略的な混沌とした問いを、構造化された役員会の議論へと変えます。ひとつの AI にひとつの滑らかな答えを求めるのではなく、鋭い知性のパネルを招き、互いに異議を唱え、意見を戦わせ、修正させ、意思決定を多角的に見渡せるよう手助けします。

これは占い師ではありません。ライフコーチでもありません。これは思考のための部屋です。

## なぜこれが存在するのか

ほとんどの重要な意思決定を妨げているのは、助言の不足ではありません。トレードオフが不明瞭であることです。

仕事を辞めるべきか？会社を立ち上げるべきか？国を移るべきか？より安全な道を選ぶべきか？リスクのあるアイデアに賭けを倍増させるべきか？お金、自由、地位、家族、好奇心、それとも長期的な選択肢の余地を選ぶべきか？

Private Board AI は、決断を下す前に交わせたらと願うような会話をシミュレートすることで、あなたを助けます：

- ある者は下振れリスクの防護の観点から論じる
- ある者は野心とレバレッジを推し進める
- ある者はあなたが本当はどんなゲームをしているのかを問う
- ある者は隠れた前提に異議を唱える
- そして役員会が議論を続けるべきかどうかは、あなたが決める

## 何ができるのか

- **パーソナルな役員会の選定**：短いプロフィールに答え、問いを投げかければ、DeepSeek があなたの状況に合った役員会を推薦します。
- **調整可能な顔ぶれ**：推薦された役員会をそのまま使う、メンバーを外す、あるいは自分でパネルを選ぶ。
- **本物の複数ラウンドの議論**：役員会はラウンドごとに分けて議論します。偽りの独白ではありません。
- **人間を介在させた継続**：2 ラウンドの後、続行するかどうかはあなたが決めます。あなたのフィードバックは次のラウンドの一部になります。
- **中国語と英語のモード**：UI、プロンプト、例、アドバイザー名、免責事項をローカライズ。
- **ローカルでの永続化**：プロフィール、問い、役員会の選択、議論の進捗はブラウザのストレージに保存されます。
- **自分の DeepSeek キーを持ち込む**：アプリはブラウザにのみ保存されるユーザー提供の API キーに対応しています。

## どんな体験になるのか

問いかける：

> 安定した仕事を辞めて、小さな AI プロダクトを作ろうかと考えています。12 か月分の貯蓄はありますが、家族は私が予測可能な道を歩むことを望んでいます。どうすべきでしょうか？

Private Board AI は単に答えるだけではありません。役員会を招集し、議論させ、そしてあなたに問いかけます：

> 彼らは続けるべきか？次に何を考慮すべきか？

あなたはこう付け加えることができます：

> 続けてください。ただし下振れリスクと、辞める前に私が検証すべきことに、より焦点を当ててください。

次の議論のラウンドは、その文脈に応えます。

## 概要

Private Board AI は「プライベート役員会」の実験です。軽量なユーザープロフィールを記入し、本当の悩みを投げかけると、システムがその問いに適した役員メンバーのグループを推薦し、あなたの問いをめぐって複数ラウンドにわたり議論させます。

AI に直接答えを告げさせるのではなく、重大な選択を、より明確なトレードオフ、リスク、機会、盲点へと分解する手助けをします。

こんな問いに向いています：

- 転職、起業、キャリアチェンジ、海外移住をすべきかどうか
- リスクは高いが上限も高い選択をすべきかどうか
- お金、自由、家族、成長、評判のあいだでどう取捨選択するか
- あるビジネス、プロダクト、キャリア、人生の意思決定が投資を続ける価値があるかどうか

| フレームワーク | Next.js 16 App Router + React 19 |

## はじめに

```bash
cp .env.local.example .env.local
```

DeepSeek API キーを追加します：

```env
DEEPSEEK_API_KEY=sk-xxx
```

依存関係をインストールし、ローカルで実行します：

```bash
npm install
npm run dev
```

開く：

```text
http://localhost:3000
```

## スクリプト

```bash
npm run dev          # start local dev server
npm run build        # production build
npm run lint         # eslint
npm test             # run tests
npm run test:watch   # watch tests
```

## プライバシーと安全性

- `.env.local` は Git によって無視され、決してコミットすべきではありません。
- ユーザー提供の API キーはブラウザに保存され、リポジトリには保存されません。
- 役員会のメンバーは、公開された考えや文体に基づく AI のシミュレーションです。
- 生成される議論は、内省と意思決定の支援のためだけのものです。法律、医療、財務、または専門的な助言ではありません。

## ステータス

これは実験的な意思決定支援インターフェースです。中核となる体験は動作しています：

- プロフィールのオンボーディング
- 役員会の推薦
- 役員会の調整
- バイリンガルのインターフェース
- 本物の複数ラウンドの議論
- ユーザー主導の継続
- 永続化された議論の状態
- DeepSeek の統合

## 誰のために作られたか

単一の答えを望まない人々のために。

重大な意思決定の前に、自分の前提に異議を唱えてほしい人々のために。

強い視点がぶつかり合うときに、よりよく考えられる人々のために。
