# Private Board AI

> 어려운 의사결정을 위한 당신만의 비공개 이사회.

Private Board AI는 복잡한 개인적 또는 전략적 질문을 구조화된 이사회 토론으로 바꿉니다. 하나의 AI에게 매끄러운 답 하나를 구하는 대신, 날카로운 두뇌들로 구성된 패널을 초대해 서로 도전하고, 의견을 달리하고, 수정하게 하며, 그 결정을 여러 각도에서 보도록 돕습니다.

이것은 점쟁이가 아닙니다. 라이프 코치도 아닙니다. 이것은 사고의 방입니다.

## 왜 존재하는가

대부분의 중요한 의사결정을 가로막는 것은 조언의 부족이 아닙니다. 불분명한 트레이드오프입니다.

직장을 그만둬야 할까? 회사를 차려야 할까? 다른 나라로 이주해야 할까? 더 안전한 길을 택해야 할까? 위험한 아이디어에 더 크게 베팅해야 할까? 돈, 자유, 지위, 가족, 호기심, 아니면 장기적인 선택지의 여지를 골라야 할까?

Private Board AI는 결정을 내리기 전에 나눌 수 있기를 바라는 그런 대화를 시뮬레이션하여 당신을 돕습니다:

- 한 사람은 하방 리스크 방어의 관점에서 논쟁한다
- 다른 한 사람은 야망과 레버리지를 밀어붙인다
- 또 다른 한 사람은 당신이 정말로 어떤 게임을 하고 있는지 묻는다
- 또 다른 한 사람은 숨겨진 가정에 도전한다
- 그리고 이사회가 토론을 계속할지 여부는 당신이 결정한다

## 무엇을 하는가

- **개인 맞춤형 이사회 선정**: 짧은 프로필에 답하고 질문을 던지면, DeepSeek이 당신의 상황에 맞는 이사회를 추천합니다.
- **조정 가능한 라인업**: 추천된 이사회를 유지하거나, 사람을 빼거나, 직접 패널을 선택할 수 있습니다.
- **진정한 다중 라운드 토론**: 이사회는 별도의 라운드로 나뉘어 토론합니다. 가짜 독백이 아닙니다.
- **사람이 개입하는 연속 진행**: 두 라운드가 끝나면, 계속할지 여부를 당신이 결정합니다. 당신의 피드백은 다음 라운드의 일부가 됩니다.
- **중국어 및 영어 모드**: UI, 프롬프트, 예시, 자문위원 이름, 면책 조항을 현지화했습니다.
- **로컬 영속성**: 프로필, 질문, 이사회 선택, 토론 진행 상황이 브라우저 저장소에 저장됩니다.
- **자신의 DeepSeek 키 사용**: 앱은 브라우저에만 저장되는 사용자 제공 API 키를 지원합니다.

## 어떤 느낌인가

질문하기:

> 저는 안정적인 직장을 떠나 작은 AI 제품을 만들까 고민하고 있습니다. 12개월치 저축이 있지만, 가족은 제가 예측 가능한 길을 가기를 기대합니다. 어떻게 해야 할까요?

Private Board AI는 단순히 답하지 않습니다. 이사회를 소집해 논쟁하게 한 뒤, 당신에게 묻습니다:

> 그들이 계속해야 할까요? 다음으로 무엇을 고려해야 할까요?

이렇게 덧붙일 수 있습니다:

> 계속하되, 하방 리스크와 그만두기 전에 제가 검증해야 할 것에 더 집중해 주세요.

다음 토론 라운드는 그 맥락에 응답합니다.

## 소개

Private Board AI는 '비공개 이사회' 실험입니다. 가벼운 사용자 프로필을 작성하고 진짜 고민을 던지면, 시스템이 그 질문에 적합한 이사진 그룹을 추천하고, 당신의 질문을 두고 여러 라운드에 걸쳐 토론하게 합니다.

AI가 직접 답을 알려주게 하는 것이 아니라, 중대한 선택을 더 명확한 트레이드오프, 리스크, 기회, 맹점으로 분해하도록 돕습니다.

이런 질문에 적합합니다:

- 이직, 창업, 직종 전환, 해외 이주를 할지 말지
- 위험하지만 상한이 높은 선택을 할지 말지
- 돈, 자유, 가족, 성장, 평판 사이에서 어떻게 취사선택할지
- 어떤 비즈니스, 제품, 커리어 또는 인생의 의사결정이 계속 투자할 가치가 있는지

## 기술 스택

| 레이어 | 선택 |
|---|---|
| 프레임워크 | Next.js 16 App Router + React 19 |
| 언어 | TypeScript |
| 스타일링 | Tailwind CSS v4 |
| LLM | DeepSeek via OpenAI-compatible API |
| 선정 모델 | `deepseek-v4-flash` |
| 토론 모델 | `deepseek-v4-flash` |
| 스트리밍 | Server-Sent Events |
| 저장소 | Browser `localStorage` + `sessionStorage` |
| 테스트 | Vitest |

## 시작하기

```bash
cp .env.local.example .env.local
```

DeepSeek API 키를 추가합니다:

```env
DEEPSEEK_API_KEY=sk-xxx
```

의존성을 설치하고 로컬에서 실행합니다:

```bash
npm install
npm run dev
```

열기:

```text
http://localhost:3000
```

## 스크립트

```bash
npm run dev          # start local dev server

npm run lint         # eslint
npm test             # run tests
npm run test:watch   # watch tests
```

## 개인정보와 안전

- `.env.local`은 Git에서 무시되며, 절대 커밋해서는 안 됩니다.
- 사용자 제공 API 키는 리포지토리가 아닌 브라우저에 저장됩니다.
- 이사회 구성원은 공개된 생각과 글쓰기 스타일에 기반한 AI 시뮬레이션입니다.
- 생성된 토론은 성찰과 의사결정 지원만을 위한 것입니다. 법률, 의료, 재무 또는 전문적 조언이 아닙니다.

## 상태

이것은 실험적인 의사결정 지원 인터페이스입니다. 핵심 경험은 작동하고 있습니다:

- 프로필 온보딩
- 이사회 추천
- 이사회 조정
- 이중 언어 인터페이스
- 진정한 다중 라운드 토론
- 사용자 주도의 연속 진행
- 영속화된 토론 상태
- DeepSeek 통합

## 누구를 위해 만들어졌나

단 하나의 답을 원하지 않는 사람들을 위해.

중대한 의사결정 전에 자신의 가정이 도전받기를 원하는 사람들을 위해.

강한 관점들이 충돌할 때 더 잘 사고하는 사람들을 위해.
